# 家計簿アプリ（Python CLI版）

このプロジェクトは、Python で作成した シンプルな家計簿アプリ（CLI版）です。  
CSV ファイルへの保存、データの追加・削除・編集、一覧表示、合計計算ができます。

今後の学習としては、以下を予定しています：

- Flask を使用した Web アプリ化  
- 関数のモジュール化（ファイル分割）  
- GUI 実装（Tkinter / CustomTkinter）
